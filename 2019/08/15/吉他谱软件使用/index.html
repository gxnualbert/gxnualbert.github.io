吉他谱软件使用

Aug 15, 2019

这两天看书，看的脑瓜疼，实在看不下去了，就练吉他。有很多的教学视频是网上的，但是没有谱子，很不方便，然后就想自己用软件来写谱子。

找了好多，都是收费的，像guitar Pro，要收199，现在很难找破解版，不像以前了。以前破解版一堆。

软件截图如下：

使用：

根据视频中的音符来使用。例如 0 0 是两个八分音符组成，那么，在软件中，先选中八分音符，然后再添加音符，最后输入数字（品格）

比如说，视频中有一个四分音符，那我们在MuseScore中，选中四分音符，然后点击菜单栏中的添加，然后在下面的六线谱中，直接点击就能添加，然后输入相应的数字，数字代表品数。

如何强制换行？？

选中需要换行的小节，然后拖动换行符到相应的小节即可。
