常用快捷键

Aug 6, 2019

Windows常用快捷键

在当前文件夹中打开命令行

有两种方法：

方法1：在地址栏中，选中地址，然后输入cmd；

方法2：在当前文件右键，选择“在此处打开命令窗口”

在Windows中，移动当前激活窗口

Win + 左箭头：移动窗口到左边，窗口 占屏幕半边大小；

Win + 右箭头：移动窗口到右边，窗口占屏幕半边大小；

Win + 上箭头： 最大化当前窗口；

Win + 下箭头：回复当前窗口；

双屏显示器中，移动窗口

Win + Shift + 左箭头： 将窗口移动到左边显示器；

Win + Shift + 右箭头： 将窗口移动到右边显示器；

快速打开任务管理器

Ctrl + Shift + Esc

关闭活动的项目或退出活动窗口

Alt + F4

最小化所有窗口

Win + m

还原最小化窗口到桌面

Win + Shift + m

打开我的电脑

Win + e

在当前目录新建文件夹

Ctrl + Shift + n

IDEA 快捷键

进入方法实现类

ctrl+alt+b

查看方法被调用

ctrl+alt+h

切换最近使用的tab

Ctrl+Tab

复制一个大括号内的内容

Ctrl+W

将当前行往上移动

alt + shift + 上箭头

将当前行往下移动

alt + shift + 下箭头

不论光标在什么位置，将光标移动到下一行

shift + enter

最大化编辑器

ctrl+shift+f12

新建一个类

Alt+Insert

将鼠标移到包上，然后Alt + Insert，可以新建类，文件夹，文件等等

方法参数提示

Ctrl + p

语句完成后，自动加分号

Ctrl+Shift+Enter

实现方法

Ctrl+i

注释/星….星/

Ctrl+shift+/

向下插入一行

shift+enter

向上插入一行

只能在空白行向上插入，非空白行的话，插入失败

ctrl+enter

选中从光标所在位置到他的父级区域

Ctrl+shift+[ 或者 Ctrl+shift+ ]

删除光标后面的单词

ctrl+del

删除光标前面的单词

ctrl+backspace

全局搜索

shift+shift

调试中，计算表达式

Alt+F8

调试中，运行到光标处

Alt+F9

调试中，切换断点

Ctrl+F8

调试中，查看断点

Ctrl+shift+F8

显示最近打开的文件

ctrl+e

跳到指定行

ctrl+g

编辑器最大化

ctrl+shift+f12, 再按一次,则返回原来的模样
